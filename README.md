# Project Documentation

## Setup Instructions

1. **Clone the repository:**
   ```bash
   git clone https://github.com/sakthee085/event_respo.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd event_respo
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Start the application:**
   ```bash
   npm start
   ```
